# IMDB Sentiment Analysis – Kaggle Project

This repository contains my solution for the Kaggle competition  
**"Bag of Words Meets Bags of Popcorn"**, which focuses on sentiment analysis of movie reviews.

The objective of the project is to classify IMDB movie reviews as **positive** or **negative** using natural language processing techniques.

## Models

Several transformer-based models were explored and evaluated:

- BERT
- RoBERTa
- DistilRoBERTa

These models were fine-tuned on the IMDB review dataset for binary sentiment classification.

## Results

The models achieved validation accuracies ranging from **0.92 to 0.95**, and prediction results were generated for Kaggle submission.
